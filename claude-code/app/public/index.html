Claude Code console

    

      
Copy: Option+drag on macOS (Shift+drag on Windows/Linux) to select and it copies automatically, or press Cmd+C / Ctrl+Insert for the selection (plain Ctrl+C still interrupts the program). The ⧉ menu grabs output straight from the terminal history with no selection — use it on phones. The wheel scrolls history. If the browser blocks the clipboard, copies land in the 📥 tray.

      
Paste: Ctrl+Shift+V / Cmd+V, or the 📋 button.

      
Attach: drag & drop files anywhere, paste an image from the clipboard, or use 📎. The file is stored and its path is typed into the prompt for Claude to read.

      
Tabs: ✳ is Claude; + opens shell tabs. Exiting Claude shows a restart menu instead of killing the session. Wheel / swipe scrolls history.

      
Update: ⬆ button, or run update-claude in a shell tab. Only the Claude session restarts — never the add-on.
